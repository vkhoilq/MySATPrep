# dashboard/vocabs/practice/

## Responsibility
SAT Vocabulary practice game modes. Implements six distinct practice types: Definition Quiz, Vocab Quiz, Vocab Match, Fill-in-the-Blank, Define (AI-evaluated), and Form a Sentence (AI-evaluated). All modes operate on learned vocabulary words from the flashcard system and track performance per word.

## Design
- **Pattern**: Practice method selection â†’ component rendering. `practice.tsx` acts as a router: user selects a method via radio group, then the corresponding practice component renders.
- **State management**: Every practice component uses `useReducer` for internal quiz state (current question, selected answer, score, answered questions, timing). All share a common `QuizState`/`QuizAction` pattern.
- **Performance tracking**: Every mode updates `PracticePerformanceData` in localStorage (`wordPerformance`, `attempts`, `strongWords`, `weakWords`, `improvingWords`). Mastery level is calculated using `correctAttempts`, `totalAttempts`, and `consecutiveCorrect`.
- **Word prioritization**: Questions are generated by categorizing learned words into performance levels (not-practiced â†’ struggling â†’ learning â†’ proficient â†’ mastered) and prioritizing less-practiced words first.

## Flow
1. User selects practice method â†’ clicks "Start" â†’ `isStarted=true` â†’ renders specific component.
2. Each component mounts â†’ loads `vocabsData` (learnt words + sentences) + `practiceData` (performance history) from localStorage.
3. `useMemo` generates questions: categorizes learned words by mastery level, shuffles within categories, prioritizes less-practiced words, builds options with same-POS distractors.
4. User progresses through questions:
   - **Quiz modes**: Select answer â†’ Submit â†’ immediate feedback (correct/incorrect) â†’ word info shown â†’ Next/Finish.
   - **Define/Sentence modes**: Type answer â†’ Submit â†’ AI validates â†’ feedback + AI analysis shown â†’ Next/Finish.
   - **Match mode**: Drag & drop â†’ immediate correctness check â†’ round complete â†’ next round or finish.
5. Each answer updates `PracticePerformanceData`: increments attempts, adjusts mastery level, categorizes into strong/weak/improving.
6. On completion: shows score, percentage, motivational message, options to restart or choose another mode.
